cordova打包vue项目成Android手机app包

使用cordova打包的好处：

Cordova是一个用基于HTML、CSS和JavaScript的，用于创建跨平台移动应用程序的快速开发平台。它使开发者能够利用iPhone、Android、等智能手机的核心功能——包括地理定位、加速器、联系人、声音和振动等，此外Cordova拥有丰富的插件，可以调用。

准备工作：

  1. Android打包环境配置：总的来说包括：jdk的安装和Java环境变量的设置、Android SDK的安装和环境变量的设置

  2. node.js的安装

  3. cordova的安装

    打包详情：

    1.初始化项目： cordova create hello com.example.hello HelloWorld

    2.切换到创建的文件夹，输入指令：cordova platform add android –save

    3.检查打包条件指令：cordova requirements

    修改vue项目

    4.首先修改vue项目的index.html

在body之间加入引入cordova.js

5在main.js里面添加vue-cordova包

6.对vue项目进行打包
参考上篇打包教程：vue-cli3.0以后版本打包问题.

7.执行完成后会生成一个dist文件夹，找到这个文件夹将里面的所有文件复制到的cordova项目的www文件夹下替换它原有的文件。

修改app的配置

修改名字：在config.xml中的标签中输入想要的名字。

图标路径：找到路径一一替换
app\platforms\android\app\src\main\res

8.然后在cordova项目终端执行：cordova build android

打包成功，会在platforms\android\app\build\outputs\apk\debug\app-debug.apk生成Android的安装包app-debug.apk

完
